Projects

Calchart 4

The Cal Band has a unique marching style, so it is difficult to design half time shows using traditional marching band charting software. Calchart 4 is the fourth iteration of Cal Band's self-made software that is used to organize the 200+ marchers on the field. I am currently leading the development of this iteration, with a focus of creating a Progressive Web App rather than a native app.

This is a completely client-side application, and I am learning a lot about front-end development. We are using Vue.js with Typescript, with emphasis on testing using Jest and Cypress. Check out the current progress at its Github repo!

michaeltamaki.github.io
